## 一般社団法人ソフトウェアエンジニアリング協会が主催する勉強会に参加した
一般社団法人ソフトウェアエンジニアリング協会が主催する勉強会とは、約３ヶ月に一度ほどの頻度で開催される勉強会で、主に東京または大阪が会場となります。
参加費は無料で、勉強会の時間は、途中休憩を挟み５時間ほどです。

コンテンツは次のような構成です。
+ 協会設立経緯および勉強会の主旨
+ 模擬面接
+ 質疑応答

端的に書いてしまうとこれだけなのですが、参加した結論としては、**非常によかった** の一言に尽きます。

ここからは、その理由について述べていきます。

本題に入る前に、まず私の経歴について簡単にご紹介します。
本協会の勉強会への参加をご検討されている方は、私の経歴を基準に、ご自身の現在の状況と照らし合わせ、適宜傾斜をかけてお読みいただければと思います。

> 筆者の経歴
・非情報学部卒
・日系大手SIerにて、インフラエンジニアの経験あり
・現職は非エンジニア
・プログラミング言語は python メイン（Findyでのスキル偏差値50程度）、C言語でlibftの再実装ができる程度
・基本情報技術者保持、応用情報は仕事との親和性が低かったため未受験

さて、本題に戻ります。

勉強会に参加して「良かった」と感じた理由は、**自分の現在位置が分かったことで、今後取るべきアクションが明確になったから**です。

勉強会のコンテンツの一つに「模擬面接」がありますが、個人的にはこの模擬面接が非常によかったと思っています。

模擬面接の面接官役を担当してくださる方は元GoogleでSWEとして働かれていた方なので、模擬面接のやり取りを見ていると、
+ SWE就職を狙うのであれば、何をどこまで想定しておくべきなのか
+ 知識ドメインとしてどのような知識があるとよいのか
+ コーディング面接で見られるところはどこなのか
といったことが肌感覚として分かります。

もちろん、一傍観者として模擬面接の応答を聞いているだけでも十分学びになりますし、自分が答える側になって、自分の実力を図る指針にするということもできるのかなと思います。

模擬面接の応答を見聞きしている過程で、その場にいるからこそ聞ける生の話もありますし、意外と完璧である必要もない、といったことも分かります。

オンラインで色々な情報が収集できてしまう現代ですが、模擬面接の課題も毎回変わる（今回はchatGPTが提案してくれたテーマの一つとのことでした）ため、模擬面接の応答から引き出される小話もおそらく異なります。どんな小話が聞けるのか、という点も魅力の一つだと思います。

とくに秘匿事項でもないので、今回の勉強会の模擬面接で取り扱った問題例を列挙します（忘れているものもあるので、全てではないです）。
+ 整数の配列が与えられるので、この整数たちから作れる連番で一番長い連番の長さを求めてください
+ 新幹線内のテザリングが遅かった理由を考えてください
+ ページビューカウンターについて（もう少し具体的な課題設定がされますが、失念しました。。）
+ オートマトンについて
+ テキストチャットサービスを作る場合のシステムデザインを考えてください

コーディング系は自分が得意な言語で、黒板に実装を書きつつ、悩む部分（入力は何で出力はどうあるべきか、etc...）を面接官に質問して、お互いのイメージを擦り合わせていくイメージです。
質問に対して質問で返される場合もあります。その場合は、返された質問が考慮すべき観点であり、現在の検討内容から漏れている点である、と捉えるとよいと思います。
あくまで、被面接者が同僚として隣の席に座っていた場合に、会話がどうなるのか、に対するイメージの解像度を上げることが目的なので、「答えられなければ落とす」という性質のものではありません。

また、黒板に書いた成果物に対しても、細かく質問が重ねられます。
その質問の内容をメモにとっておくだけでも、自身の現在地を知る上では非常に有意義かと思います。

その後の質疑応答も参考になることが多いかなと思います。
質疑応答もとい人生相談なので、質問内容も幅広いです。これだけはぜひ聞きたい、というものがあれば、ぜひ全体の場で聞いてみるとよいと思います。

もし、質問がたくさんある、という方や、質問の内容が全体の場で質問する内容としてはそぐわない、という場合は、勉強会後の懇親会の場で直接理事の方に質問してみるとよいと思います。
本当に何でも答えていただけます。

筆者は、どこかのタイミングで海外就職をしたいと考えていますが、目的によってとるべき戦略が異なり、どのような目的に対してどのような戦略をとるべきなのか、に対する解像度も100倍に高まるといっても過言ではありません。

筆者も懇親会の場でたくさん質問をさせていただきましたが、今後とるべきアクションのリストが20個程度出てきたかな、という体感です。

「SWE就職を目指しているけど、何をすべきか分からない」
「SWE就職を目指しているけど、自分の実力を確かめたい」

といった方は、ぜひ勉強会に参加してみてください。
何かしらの学びは得られると思います。
